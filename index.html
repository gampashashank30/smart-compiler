                  AI Root Cause

                
Uninitialized stack variable used in
                  accumulation.

                

                  In C, stack variables are not initialized to 0. They hold whatever residual garbage values were stored
                  at that memory slot previously.
                

              

            


            
            

              
AI Fix Suggestion

              

                

                  

                    - int total;
                  

                  

                    + int total = 0;
                  

                

                

                  Setting total = 0 ensures you start accumulating from
                  a clean, known value.
